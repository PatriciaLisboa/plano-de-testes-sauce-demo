# Plano de testes — Sauce Demo

Repositório de estudos de QA Manual com foco em documentação de testes
para a aplicação [Sauce Demo](https://www.saucedemo.com), uma loja virtual
criada especificamente para prática de testes de software.

## Objetivo

Desenvolver e documentar artefatos de QA como plano de testes, casos de
teste e bug reports, simulando o trabalho real de uma analista de QA Manual.

## Aplicação testada

- **Nome:** Sauce Demo
- **URL:** https://www.saucedemo.com
- **Tipo:** E-commerce fictício para prática de testes

## Ferramentas utilizadas

- Documentação: Markdown
- Versionamento: Git e GitHub
- Navegador: Firefox
